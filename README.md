# vantd-router-demo

一个简单的 vue-router 多页签的解决方案 demo，UI 用的 antd
由于用了 window.atob ,window.btoa 等方法，故不支持 ie9 及以下浏览器，想要用的自己去搜 polyfill 解决方法

## Project setup

```
npm install
```

### Compiles and hot-reloads for development
